## 🗂️ Description
A desktop application that notifies users of upcoming password expiration through a simple GUI

## 🤖 Tech Stack
PowerShell

## ✏️ Key Functionality
- Automated user notifications to prompt password changes before password expiration using PowerShell.

## 👍 Benefits
- Reduced IT workload and prevented account lockouts by automating password expiration notifications.
